## 概览
该日志 CSV 共 243 列，结构明显是“机器人/机械臂控制器调试日志”：
- 3 列时间相关：ProgramTime / PlanTime / NodeTime
- 4 列节点/任务相关字符串：NodeName / NodePath / PTName / SubNodePTName / ToolName
- 大量按编号展开的时序信号：Cart*（笛卡尔）、Jnt*（关节）、Motor*（电机/驱动）、Wrench*（力/力矩）、Torque*（力矩估计/观测）等

本文件的列名采用固定命名规则（工业常见的“前缀 + 通道编号 + 量名/来源”）。因此“理解每一列”最稳妥的方式是建立**规则化字典**，既能覆盖当前文件，也能覆盖同系列日志的“所有标题”。

---

## 1. 时间与节点信息
| 列名 | 类型 | 建议单位/语义 |
|---|---|---|
| ProgramTime | number | 程序时间（通常是秒级时间轴或控制器内部时间；前端会自动识别并转为 ms 供绘图） |
| PlanTime | number | 规划周期/规划时间（单位与控制器实现相关，常见为 s） |
| NodeTime | number | 节点内部耗时/周期（单位与控制器实现相关，常见为 s） |
| NodeName | string | 当前执行节点名称（用于分段/筛选/标注） |
| NodePath | string | 节点路径（用于分段/筛选/标注） |
| PTName | string | PT（过程/任务）名称（用于分段/筛选/标注） |
| SubNodePTName | string | 子节点 PT 名称（用于分段/筛选/标注） |
| ToolName | string | 工具/末端执行器名称（用于分段/筛选/标注） |

## 2. Cart（笛卡尔空间：末端/工具位姿、速度、期望与误差）
Cart 系列有 6 个通道（0..5），通常对应：
- 0..2：位置 X/Y/Z（m）
- 3..5：姿态 RX/RY/RZ（rad，或控制器内部的旋转参数；在该日志中数值范围更像欧拉角/旋转向量的某种展开）

| 命名规则 | 语义 | 单位（建议） |
|---|---|---|
| Cart{i}CurPos | 当前笛卡尔位姿分量 | i=0..2: m；i=3..5: rad |
| Cart{i}CurVel | 当前笛卡尔速度分量 | i=0..2: m/s；i=3..5: rad/s |
| Cart{i}DesiredPos | 期望笛卡尔位姿分量 | 同上 |
| Cart{i}DesiredVel | 期望笛卡尔速度分量 | 同上 |
| Cart{i}PosErr | 位姿误差分量（Desired - Cur） | 同上 |
| Cart{i}VelErr | 速度误差分量（Desired - Cur） | 同上 |

---

## 3. Jnt（关节空间：命令/反馈/误差/力矩）
Jnt 系列有 7 个关节通道（0..6），常见对应 7-DOF 机械臂。

| 命名规则 | 语义 | 单位（建议） |
|---|---|---|
| Jnt{j}CurPos | 当前关节位置 | rad |
| Jnt{j}CurVel | 当前关节速度 | rad/s |
| Jnt{j}CurAcc | 当前关节加速度 | rad/s² |
| Jnt{j}PosCmd | 位置命令 | rad |
| Jnt{j}VelCmd | 速度命令 | rad/s |
| Jnt{j}AccCmd | 加速度命令 | rad/s² |
| Jnt{j}FdPosCmd / Jnt{j}FdVelCmd / Jnt{j}FdAccCmd | 前馈（feedforward）命令 | 同上 |
| Jnt{j}PDVelCmd | PD 控制输出的速度命令（或中间量） | rad/s |
| Jnt{j}PosErr | 位置误差 | rad |
| Jnt{j}VelErr | 速度误差 | rad/s |
| Jnt{j}CurTorque | 当前关节力矩（估计/测量） | N·m |
| Jnt{j}TorqueCmd | 力矩命令 | N·m |
| Jnt{j}TorqueFF | 力矩前馈 | N·m |

## 6. Wrench（六维力/力矩：期望、由关节力矩反推、由传感器测量）
Wrench 系列有 6 个通道（0..5），通常对应：
- 0..2：力 Fx/Fy/Fz（N）
- 3..5：力矩 Tx/Ty/Tz（N·m）

| 命名规则 | 语义 | 单位（建议） |
|---|---|---|
| Wrench{k}Desired | 期望六维力/力矩分量 | k=0..2: N；k=3..5: N·m |
| Wrench{k}FromJntTorque | 由关节力矩反推得到的六维力/力矩分量 | 同上 |
| Wrench{k}FromSensor | 由六维力传感器得到的六维力/力矩分量 | 同上 |

## 11. Manipulability（可操作性指标）
Manipulability 有 3 组（0..2），每组包含：
- ValueTrans：平移可操作性
- ValueOri：姿态可操作性

| 命名规则 | 语义 | 单位（建议） |
|---|---|---|
| Manipulability{i}ValueTrans | 平移可操作性指标 | 无量纲 |
| Manipulability{i}ValueOri | 姿态可操作性指标 | 无量纲 |

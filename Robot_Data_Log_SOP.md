# 机器人日志导出及分析平台 (Robot Project Data in Log) SOP

本文档为机器人日志导出与 **Robot Project Data in Log** 分析平台的操作标准流程 (SOP)，涵盖了从机器人软件中导出原始日志文件，到将数据导入本平台并进行可视化分析的完整步骤。

---

## 🔧 第一部分：从机器人导出日志 (以 Flexiv 为例)

1. **进入机器人软件**：打开 Flexiv Elements Studio 客户端。
2. **更改连接配置**：连接到机器人，选择并进入 `flexivconn`。
3. **进入专家模式**：在登录/权限界面输入专家密码：`flexiv`。
4. **进入日志目录**：在系统菜单中找到并进入机器人的 `log` 文件目录。
5. **选择并导出日志**：
   - 按照文件名称或修改时间排序。
   - 选择对应时间节点的 log 日志（*.csv 格式）并将其**导出**到本地电脑。
   - *(注：该内部 log 文件为控制箱重启一次生成一次，请确保导出你需要分析的那段时间的日志。)*

---

## 📥 第二部分：将日志导入分析平台

1. **访问分析平台**：
   - 打开部署好的平台网址（如：`https://robot-project-data-log.pages.dev/`）。
2. **上传日志文件**：
   - **方式一（推荐）**：直接将刚导出的 `.csv` 日志文件**拖拽**到页面中央的上传卡片区域。
   - **方式二**：点击页面中央的“**选择文件并开始分析**”按钮，在弹出的文件管理器中找到并选中该 `.csv` 文件。
3. **等待解析**：
   - 平台将自动识别并解析文件，显示带有蓝色环形进度条的“正在读取文件...”及“正在解析数据...”。
   - 解析完成后（图标变为绿色对勾），系统会自动跳转进入主工作台界面。

---

## 📊 第三部分：数据筛选与波形分析

### 1. 顶部全局过滤 (缩小分析范围)
进入主工作台后，可以通过顶部下拉菜单对数据进行**整体截取**：
- **NodeName (节点名称)** / **NodePath (节点路径)**
- **PTName (规划任务名称)** / **SubNodePTName (子节点任务)** / **ToolName (工具名称)**
- 根据需要选择特定条件，平台会自动切除无关数据，只留下你选定条件内的日志段落。

### 2. 左侧传感器列表与面板 (Pane) 管理
- **快速检索**：在左侧“传感器列表”的搜索框中输入关键字（如 `pose_x`、`imu_az`），快速查找你需要分析的底层信号。
- **面板 (Pane) 分组**：
  - 点击列表右上角的 **`+`** 号图标，可创建多个独立的 Pane（面板），用于在右侧分块对比查看不同类别的信号（例如：Pane 1 看力矩，Pane 2 看位置）。
  - 若需删除某一组，选中对应 Pane 标签后点击 **`-`** 号即可。
- **调整视野**：鼠标放在左侧列表与右侧波形图中间的间隙处，可按住**左右拖拽**来调整侧边栏的宽度。

### 3. 查看波形与极值分析
- **绘制波形**：在左侧列表中勾选你感兴趣的传感器字段（如 `CART` 笛卡尔位姿、`JNT` 关节空间等），右侧主视图会立刻绘制出该信号随时间变化的**时间序列波形趋势图**。
- **波形交互**：
  - **缩放**：在图表内滚动鼠标滚轮，可对波形进行 X/Y 轴的自由缩放。
  - **平移**：在图表内按住鼠标左键并拖动，可平移查看不同时间段的波形。
  - **精准数值**：将鼠标悬停在波形的某个点上，会弹出该时间点精准的数值提示框。
- **统计面板**：波形图上方的大字面板会实时计算并展示当前选中信号的四大核心指标：
  - **数据总数 (行)**
  - **平均数值**
  - **最大峰值**
  - **最小谷值**

---

> **💡 常见问题与技巧**
> - **为什么导入太慢？** 平台基于浏览器本地算力，若文件达到几百 MB，首次读取会稍有延迟，请耐心等待进度条走完。
> - **找不到想要的信号？** 善用左侧的搜索框，部分信号可能折叠在 `MOTOR` 或 `WRENCH` 等父级目录下，展开即可查看。
